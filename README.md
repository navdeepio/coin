# Usage
<!-- usage -->
```sh-session
$ npm install -g coincli
$ coin COMMAND
running command...
$ coin (-v|--version|version)
coincli/0.0.1 linux-x64 node-v11.1.0
$ coin --help [COMMAND]
USAGE
  $ coin COMMAND
...
```
